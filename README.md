# Ansible LEMP Stack Setup on Amazon Linux

This project automates the installation and configuration of a **LEMP stack (Linux, Nginx, MySQL, PHP)** on **Amazon Linux EC2 instances** using **Ansible**.  
It is designed following Ansible best practices and is suitable for **real-world DevOps usage and interviews**.
